# Xtal Backend

# Features

# Built With

- TypeScript
- Express

# Setup

1. Create an `.env` file at backend root directory (`/backend`) with the following properties:

```shell
PORT=3000
```

# Running the application

1. Navigate to the `/backend` directory and run `npm run dev`
